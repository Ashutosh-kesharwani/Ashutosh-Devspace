# 🌐 Personal Portfolio Website

A modern, interactive, and performance-focused **developer portfolio** showcasing my work, skills, and design-to-code approach. Built with clean front-end practices, smooth animations, and a strong focus on UI/UX.


---



## ✨ Highlight



* ⚡ Smooth scroll-based animations (GSAP)
* 🎨 Design → Code focused layout
* 🖱️ Custom cursor interactions
* 📱 Fully responsive across devices
* 🧠 Clean, readable, and scalable code

---

## 🛠️ Tech Stack

* **HTML5** – Semantic markup
* **CSS3** – Modern layouts & animations
* **JavaScript (ES6+)** – Interactivity & logic
* **GSAP** – Animations & transitions
* **Locomotive Scroll** – Smooth scrolling experience

---

## 📂 Sections

* **Landing / Hero** – Introduction & first impression
* **About** – Background and design philosophy
* **Design → Code** – Principles & approach
* **Projects** – Selected works with interactions
* **Contact** – Ways to connect

```bash

```

Open `index.html` in your browser or use a local server for best experience.
